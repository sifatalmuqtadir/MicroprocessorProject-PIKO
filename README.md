# PIKO Robot

Microprocessor-Project, PIKO is a smart Arduino-based robot that combines:

* Obstacle Avoidance
* Voice Control
* Bluetooth Control

## Components

* Arduino Uno
* HC-SR04 Ultrasonic Sensor
* HC-05 Bluetooth Module
* Voice Recognition Module
* L298N Motor Driver
* IR Sensors
* SG90 Servo Motor

## Features

* Autonomous navigation
* Voice command control
* Smartphone control via Bluetooth
* Real-time mode switching
* Emergency stop functionality
